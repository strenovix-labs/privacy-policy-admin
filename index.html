Privacy Policy — VoteAble

Effective Date: March 24, 2026

Google Play Compliant

1. About VoteAble

VoteAble is a secure election management platform used by authorized officers such as BLO, RO, and ARO. It enables voter assistance tracking, real-time mapping, and election coordination.

2. Data We Collect

  • Name, Phone Number, EPIC ID
  • Gender, Age, DOB
  • Location (Latitude & Longitude)
  • Address (Taluk, District, Pincode)
  • Disability Type & Assistance Needs
  • Voting Intent

3. How We Use Data

  • Map voters and assistance requirements
  • Show DAP persons on map
  • Enable officers to locate voters
  • Manage and resolve queries
  • Provide dashboards and analytics

4. Location Usage

Real-time location of officers is used only when the app is active to show nearby voters needing assistance. No background tracking is performed.

5. Third-Party Services

  • Google Maps API — map rendering
  • Firebase — push notifications
  • Railway — backend hosting
  • PostgreSQL — database storage

6. Data Sharing

We do not sell or share data with advertisers or third parties. Data is accessible only by authorized election officials.

7. Permissions

  • Location — for map features
  • Internet — backend communication

8. Security

  • HTTPS encryption
  • Secure PostgreSQL database
  • Role-based access control

9. Children's Privacy

This app is intended only for users above 18 years.
